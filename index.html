Welcome

Have you ever wanted to know what time it definitely isn't?

Now you can! Using my amazing algorithm (that I spent two minutes on), this website will always display the incorrect time.

And just in case you thought it could maybe be helpful once, for every real world minute, The World's Worst Clock will automatically change to a completely random time, ensuring an incorrect reading every time.

You know that saying "A broken clock is right twice a day?" Well, unfortunately for you, I made sure that was impossible.

From an entire 23 hour difference to just being off by just a minute, The World's Worst Clock is dedicated to giving you false information and have a small but noticible negative impact in your day.

Watch in amazement as the seconds count up before switching between completely useless information to new information that is different but equally unhelpful.

Whether you're bored and willing to watch anything, have literally nothing else better to do, or just want to defy Father Time in a futile attempt to resist the natural order of the universe like the heretic you are, I welcome you.

Have fun! I literally don't know how you would, but have fun!

FAQ

Q: How does the Clock Work?

When you open the webpage, it connects directly to your system clock and automatically changes the time by a random amount within 1 minute and 23 hours and 59 minutes, making it mathmatically impossible for it to display the correct time.

This guarantees that this clock will provide absolutely meaningless information 100% of the time.


Q: What's the coding behind the Clock?

The clock uses JavaScript's Built-In Math Class to provide a random value via the Random method and adds it to an element containing the current time, created using JavaScript's API, which accesses the system's current time. Then, by setting a global timeout delay, the function is set to run every 1000 miliseconds, constantly rechecking the date element and uses an if statement to determine whether a minute as passed, to which it re-randomizes the value added ensure a different randomized value every minute.
